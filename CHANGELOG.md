# Changelog

All notable changes to the Emblem Tabletop System will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1-alpha] - 2025-11-29

### Added
- **Comprehensive Magic Triangle System**: Implemented two-level weapon triangle for magic
  - Level 1 (Primary): Light > Dark > Anima > Light (+15 hit, +1 damage)
  - Level 2 (Anima Subtypes): Thunder > Fire > Wind > Thunder (+10 hit, +1 damage)
- Magic system now properly handles all elemental relationships
- Dark magic has advantage over all Anima subtypes (fire, thunder, wind)

### Removed
- Visual weapon triangle displays from character sheets (mechanics still active in combat)

## [0.1.0-alpha] - 2025-11-29

### Initial Release
- Basic character and enemy actor sheets
- Weapon, skill, class, and item management
- Combat system with interactive resolution
- Tactical grid movement and attack range visualization
- Weapon durability tracking
- Physical weapon triangle (Sword > Axe > Lance)
- Stat growth and level-up system
- Dual language support (English/Spanish)
- Complete rebrand to "Emblem" with legal disclaimers
- MIT License with fan-project notices
