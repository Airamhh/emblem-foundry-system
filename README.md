# Emblem Tabletop System

?? **ALPHA STATUS** - This project is in early alpha development. Expect bugs, incomplete features, and breaking changes.

## ?? What is This?

An unofficial, fan-made tactical RPG system for Foundry Virtual Tabletop, inspired by classic strategy games and based on the community TTRPG adaptation by Dylan B.

## ?? Quick Start

1. Create a Character actor
2. Set stats and growth rates  
3. Add weapons to inventory
4. Equip a weapon
5. Create Enemy actors
6. Right-click token ? "Preview Combat" to fight

## ?? Documentation

Documentation is minimal. System is based on Dylan B.'@ + "'" + @'s manual (see link above).

**Key Mechanics:**
- Stats: HP, Str, Mag, Skl, Spd, Lck, Def, Res, Mov
- Weapon types: Sword, Lance, Axe, Bow, Tome, Staff, etc.
- Weapon Triangle: Sword > Axe > Lance > Sword
- Combat: Auto-calculated hit/damage/crit with chat output
- Growth: Roll % chance per stat on level-up
